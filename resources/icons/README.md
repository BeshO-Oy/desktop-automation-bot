# Icon Templates Directory

## How to Add Notepad Icon Template

1. **Capture the Notepad icon:**
   - Take a screenshot of just the Notepad icon on your desktop
   - The icon should be clear and visible
   - Recommended size: 64x64 pixels or larger

2. **Save the image:**
   - Save it as: `notepad_icon.png` in this directory
   - Supported formats: PNG, JPG, BMP

3. **Tips for best results:**
   - Capture the icon when desktop is visible
   - Use PNG format for best quality
   - Make sure the icon is not obscured
   - Capture at the same icon size you're using (small/medium/large)

## File Structure

```
resources/
  icons/
    notepad_icon.png  <- Put your Notepad icon screenshot here
```

## Usage

Once you add `notepad_icon.png`, the bot will automatically use it for more accurate detection.

If no template is provided, the bot will use the flexible detection methods (which may detect other icons).
